◆ decomposicaoLU()

std::tuple< Matriz< double >, Matriz< double >, Matriz< double > > ifnum::decomposicaoLU ( const Matriz< double > &  A)

Decompõe uma matriz quadrada em L, U e P com pivotamento completo.

A função realiza a decomposição LU com pivotamento completo, retornando:

  • L: matriz triangular inferior,
  • U: matriz triangular superior,
  • P: matriz de permutação.
Parâmetros
AMatriz quadrada a ser decomposta.
Retorna
Tupla contendo as matrizes L, U e P, respectivamente.
Exceções
std::runtime_errorSe a matriz for singular.

Matriz< T > &  b

Aplica o método de eliminação de Gauss para escalonar uma matriz.

Realiza a triangularização superior de uma matriz de coeficientes e modifica o vetor de constantes correspondente. Não realiza substituição para encontrar a solução — apenas escalona o sistema.

Parâmetros
matrizMatriz dos coeficientes (modificada no processo).
bVetor de constantes do sistema (modificado no processo).
Retorna
Matriz escalonada.
Exceções
std::invalid_argumentSe a matriz não for quadrada.

◆ resolverLU()

Matriz< double > ifnum::resolverLU ( const Matriz< double > &  A,
const Matriz< double > &  b 
)

Resolve um sistema linear utilizando a decomposição LU com pivotamento completo.

Resolve o sistema ( Ax = b ) através de:

  1. Decomposição LU com pivotamento completo.
  2. Substituição direta e reversa para encontrar o vetor solução.
Parâmetros
AMatriz dos coeficientes (matriz quadrada).
bVetor coluna com os termos independentes.
Retorna
Vetor solução do sistema.
Exceções
std::runtime_errorSe a decomposição não for possível (matriz singular).
