template<typename T>
class ifnum::Vetor< T >

Classe que representa um vetor matemático genérico.

Armazena as coordenadas do vetor e fornece operações básicas como soma, subtração, normalização, produto escalar e comparação.

Parâmetros do template
TTipo dos elementos do vetor (ex: int, float, double).

Construtor que inicializa o vetor com um tamanho fixo e dados fornecidos.

Copia os elementos de dados para o vetor até o limite de tamanho. Se dados tiver menos elementos, o restante será preenchido com valores padrão de T. Caso dados seja vazio, o vetor será totalmente preenchido com valores padrão.

Parâmetros do template
TTipo dos elementos do vetor.
Parâmetros
tamanhoTamanho final do vetor.
dadosVetor com os dados a serem copiados (opcional).
